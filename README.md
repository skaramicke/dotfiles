# dotfiles

This package is meant for use with zsh on macos. It is a collection of dotfiles and scripts that I use to make my life easier.

## Add tags to markdown files

```zsh
add_md_tags.sh <files, i.e. docs/*.md> <tag1> <tag2> <tag3> ...
```
